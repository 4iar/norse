Norse v0.0.2
===================

Norse is a `deep-learning <https://en.wikipedia.org/wiki/Deep_learning>`_ library
for `spiking neural networks
<https://en.wikipedia.org/wiki/Spiking_neural_network>`_.
Norse is built on `PyTorch <https://pytorch.org/>`_, which makes it easy to use,
fast, and robust.
