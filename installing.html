2. Installing Norse¶

We have chosen to build Norse with new features such as type hints. For that reason we require Python version 3.7 or higher. If this is a problem, it is recommended to install Norse from a Docker image using the correct Python version.

Norse builds on top of the PyTorch deep learning library, which is also our primary dependency. This has the benefit that your models are hardware accelerated, providing the prerequisites are met.

For that reason, we recommend following the PyTorch “Get Started” guide as the first step.

After installing the Torch prerequisite, the primary dependencies can be done in the following three ways:

2.1. Installing from PyPi¶

This requires command-line access to

pip install norse

  • Installing from Conda

conda install -c norse norse

  • Installing from source

git clone https://github.com/norse/norse
cd norse
pip install -e norse

2.2. Optional dependencies¶

Norse runs well with the above installation, but some of the tasks integrates with the visualization tool Tensorboard to make it easier to introspect models and results.
